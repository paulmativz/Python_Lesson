5. Variables
Like a container - nilalagyan ng value
When creating variable names in python, meron tayong mga variable name rules na kailangan sundin:
Ang isang variable name ay pwede lang mag-umpisa sa isang letter or underscore
Hindi pwedeng mag-umpisa ang variable name mo sa isang number
Pwedeng maglagay ng number and any characters after the first letter.
Variable names are case-sensitive

Variable can only hold 1 value

7. Numeric Operators
 Addition operator = +
   x = 24
   y = 8
   sum1 = x + y
   sum2 = 24 + 8
   print(sum1)
   print(sum2)
   print(x + y)
   print(8 + 24)
   
 Subtraction Operator = -
    x = 24
    y = 8
    diff1 = x - y
    diff2 = 24 - 8
    print(diff1)
    print(diff2)
    print(x - y)
    print(24 - 8)
	
 Multiplication Operator = *
   x = 24
   y = 8
   prod1 = x * y
   prod2 = 24 * 8
   print(prod1)
   print(prod2)
   print(x * y)
   print(24 * 8)
